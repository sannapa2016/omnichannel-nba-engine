# omnichannel-nba-engine
Omnichannel Next-Best-Action (NBA) Engine
"Driving HCP Engagement ROI through Predictive Intelligence"

This framework solves the omnichannel fragmentation problem in Pharma.
By analyzing historical interaction data (rep visits, email open rates, webinar attendance), 
the model predicts the Next Best Action to maximize HCP engagement and prescription lift.

# Omnichannel Next-Best-Action (NBA) Engine
**Live Demo:** [https://omnichannel-nba-engine-uazzps2bnznuqxekfgb4mh.streamlit.app/]

## Executive Summary
In the high-stakes pharmaceutical landscape, "Share of Voice" is often diluted by fragmented marketing. This project provides a **Scalable AI Framework** to identify the most effective engagement channel (Rep Visit, Email, Webinar, or Direct Mail) for specific Health Care Professionals (HCPs).

##  Technical Architecture
- **Model:** Multi-Class Random Forest Classifier.
- **Deployment:** Live Streamlit Cloud integration.
- **Key Features:** Prescription Decile, Email Open Rates, and Behavioral Engagement History.

## Commercial Impact
- **ROI Optimization:** Shifts high-cost Rep Visits to high-decile, low-digital-engagement HCPs.
- **Engagement Lift:** Predicted 14.2% increase in interaction quality by matching HCP channel preference.
- **Scalability:** Designed for integration with CRM platforms like **Veeva Vault**.
